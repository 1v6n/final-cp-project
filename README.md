# Trabajo Practico Final - Programación Concurrente

## Red de Petri / Sistema de Agencia de Viajes

Este proyecto en Java implementa un sistema de gestión de una agencia de viajes modelado mediante una Red de Petri, 
simulando flujos de clientes y gestión de reservas con énfasis en la programación concurrente para la eficiente 
administración de recursos compartidos. Se desarrolla un monitor de concurrencia para controlar la ejecución y 
el disparo de transiciones y se optimiza la gestión de hilos basándose en el análisis de invariantes de transición 
para maximizar el paralelismo.
